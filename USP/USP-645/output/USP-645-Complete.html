1. Introduction and Instrument Specifications — 導論與儀器規格

原文內容 Original Text

Electrical conductivity in water is a measure of the ion-facilitated electron flow through it. Water molecules dissociate into ions as a function of pH and temperature and result in a very predictable conductivity. Some gases, most notably carbon dioxide, readily dissolve in water and interact to form ions, which predictably affect conductivity also.

Water conductivity is also affected by the presence of extraneous ions. The extraneous ions used in modeling the conductivity specifications are the chloride and ammonia ions. The conductivity of the chloride ion (at 0.47 ppm) and the ammonium ion (at 0.3 ppm limit) represents a major portion of the allowed water ionic impurity level.

conductivity, κ (S/cm) = Θ (cm⁻¹) / R (Ω)

where Θ = cell constant, R = resistance

Cell Constant (Θ): Must be known within ±2%. Verified directly using a solution of known conductivity, or indirectly by comparing to a sensor of known cell constant.

Resistance Measurement: Use precision resistors traceable to NIST (accurate to ±0.1%). Measured conductivity must be within ±0.1 µS/cm of calculated value. Minimum resolution: 0.1 µS/cm on the lowest range.

Temperature Compensation: Values used in the Stage 1 test for Bulk Water are non–temperature-compensated measurements. Accuracy of temperature measurement must be ±2°.

導師解析 Tutorial Commentary

核心概念解析 Key Concepts

電導率 (conductivity) 測量的是水中離子的總量。製藥用水的電導率主要由以下因素決定：

  • 水自身解離：H₂O → H⁺ + OH⁻，受 pH 和溫度影響
  • 溶解 CO₂：CO₂ + H₂O → H₂CO₃ → HCO₃⁻ + H⁺（顯著增加導電率）
  • 外來離子 (extraneous ions)：Cl⁻（氯化物）和 NH₄⁺（銨根）是模型中的代表污染物

三階段測試的設計讓大多數合格樣品在 Stage 1 即可通過，只有邊際 (borderline) 樣品才需要進行更繁瑣的後續測試。

公式與計算 Formula

電導率計算：
κ (S/cm) = Θ (cm⁻¹) / R (Ω)

其中：
  Θ = 電池常數 (cell constant, d/A)
      d = 電極間距離 (cm)
      A = 電極面積 (cm²)
  R = 電阻 (Ω)

校正要求：
  電池常數 Θ 誤差 ≤ ±2%
  電阻測量精度：±0.1% (NIST traceable)
  測量結果誤差 ≤ ±0.1 µS/cm
  最低解析度：0.1 µS/cm

溫度測量精度：±2°C

比喻說明 Analogy

電導率儀就像水的「離子計數器」：每個溶解在水中的離子都像一個微小的電池，帶著電荷在電場中移動。離子越多，電流越易流通，導電率越高。純化水因為離子極少，導電率接近純水的理論最低值。

電池常數 (cell constant) 相當於「儀器的校正比例尺」——不同的探頭幾何形狀需要不同的修正係數，就像不同的量尺需要確認其刻度是否準確。

重點提示 Key Notes

為何 Stage 1 禁用溫度補償？溫度補償演算法 (temperature compensation algorithm) 是基於特定離子類型建立的模型，對純化水中複雜的離子組合可能不準確。USP 要求 Stage 1 使用未補償的原始讀數 (non-temperature-compensated)，然後與相應溫度的查詢表 (Table 1) 直接比較，避免補償誤差帶來的假陰性或假陽性。

2. Bulk Water: Three-Stage Test — 散裝水三階段測試

原文內容 Original Text

This is a three-stage test method to accommodate online or offline testing. The procedure can be started at Stage 2 if offline testing is preferred.

STAGE 1 (online or offline):

  1. Determine temperature and non–temperature-compensated conductivity.
  2. Using Table 1, find the temperature value NMT the measured temperature. The corresponding conductivity is the limit. [Note—Do not interpolate.]
  3. If measured conductivity ≤ table value: passes. If higher: proceed to Stage 2.

Table 1 (selected values) — non-temperature-compensated measurements only.

STAGE 2 (offline, equilibrated):

  1. Transfer to suitable container. Maintain at 25 ± 1°C. Vigorously agitate while observing conductivity. When change < 0.1 µS/cm per 5 min, note conductivity. [May be temperature-compensated to 25°.]
  2. If conductivity ≤ 2.1 µS/cm: passes. If > 2.1 µS/cm: proceed to Stage 3.

STAGE 3 (pH equilibration):

  1. Within ~5 min of Stage 2. Add saturated KCl (0.3 mL per 100 mL). Determine pH to nearest 0.1 unit per USP 〈791〉.
  2. Refer to Table 2. If measured conductivity (from step 4) ≤ table value at measured pH: passes. If conductivity > table value OR pH outside 5.0–7.0: fails.

導師解析 Tutorial Commentary

核心概念解析 Key Concepts

三階段測試的設計邏輯反映了電導率超標的可能原因：

  • Stage 1（快速篩查）：與溫度相關的限度表直接比對，大多數合格樣品在此通過
  • Stage 2（大氣 CO₂ 平衡）：讓大氣中 CO₂ 充分溶解達到平衡，排除 CO₂ 不足造成的假陽性；限度 2.1 µS/cm 是 25°C 下純水吸收大氣 CO₂ 後的預期最大值
  • Stage 3（pH 確認）：加入 KCl 平衡後測 pH，確認電導率超標是否由 pH 異常離子（酸或鹼）引起；提供 pH 5.0–7.0 範圍內的精確限度

三階段測試流程與關鍵數值

STAGE 1（線上監測，非溫度補償）：
  查表 Table 1，取「不超過測量溫度」的值
  25°C → 限度 1.3 µS/cm
  40°C → 限度 1.7 µS/cm
  [禁止內插！直接取下一個較低溫度的限度值]

  合格 → 結束
  不合格 → 進入 Stage 2

STAGE 2（25°C 平衡，可溫度補償）：
  限度：2.1 µS/cm（固定值，所有水型）
  合格 → 結束
  不合格 → 進入 Stage 3

STAGE 3（加 KCl，測 pH）：
  pH 範圍：5.0–7.0
  查表 Table 2 對應 pH 的限度
  pH 5.0 → 4.7 µS/cm
  pH 6.0 → 2.4 µS/cm（最低點）
  pH 7.0 → 4.6 µS/cm
  pH 在 5.0–7.0 外 → 直接不合格

比喻說明 Analogy

三階段測試就像給病人看診：Stage 1 是快速體溫測量（快速篩查，大多數人通過）；Stage 2 是標準血壓測量（在標準環境下的基準測試）；Stage 3 是抽血化驗（最詳細的確認，找出具體問題）。三個階段由快到慢、由粗到精，讓大多數正常樣品不必做昂貴費時的後續測試。

重點提示 Key Notes

Stage 1「不得內插」的重要性：若水溫為 27°C，Table 1 中沒有 27°C 的數值，應使用「不超過 27°C」的最近溫度值，即 25°C 對應的 1.3 µS/cm 作為限度。不得在 25°C 和 30°C 的數值之間內插，因為這會引入系統性誤差。

Stage 2 的操作細節：「劇烈攪拌 (vigorously agitate)」是為了加速 CO₂ 的大氣溶解過程；「每 5 分鐘變化 < 0.1 µS/cm」是平衡達成的判斷標準。若使用密閉容器，CO₂ 無法充分溶解，Stage 2 測試結果將無效。

實務應用 Practical Application

在無菌製造 CDMO 的 WFI 系統中，通常在分配迴路 (distribution loop) 關鍵點安裝線上電導率儀，連續監測並設定警戒限 (alert limit, 例如 1.0 µS/cm) 和行動限 (action limit, 例如 1.3 µS/cm)。當警戒限被觸發時，系統自動記錄並提醒 QC，但生產繼續；當行動限被觸發時，自動隔離受影響的水批次，啟動偏差調查 (deviation investigation)。Stage 1 查表法是這種線上監測的法規依據。

3. Stage 3 Table and Sterile Water — Stage 3 pH 對照表與無菌水

原文內容 Original Text

Table 2. Stage 3—pH and Conductivity Requirements (for atmosphere- and temperature-equilibrated samples only)

pH outside 5.0–7.0: water does not meet requirements.

STERILE WATER:

Applies to: Sterile Purified Water, Sterile Water for Injection, Sterile Water for Inhalation, Sterile Water for Irrigation.

Procedure:

  1. Obtain sample; vigorously agitate package before opening.
  2. Transfer to suitable container, stir. Maintain at 25 ± 1°C.
  3. Agitate while observing conductivity; when change < 0.1 µS/cm per 5 min, note conductivity.

Acceptance Criteria:

  • Containers ≤ 10 mL nominal volume: NMT 25 µS/cm
  • Containers > 10 mL nominal volume: NMT 5 µS/cm

導師解析 Tutorial Commentary

核心概念解析 Key Concepts

Stage 3 Table 2 的「V 形曲線」揭示了一個重要的化學現象：

  • pH < 6.0（偏酸性）：H⁺ 離子濃度上升 → 電導率升高 → 限度放寬
  • pH 6.0–6.6（最純範圍）：接近中性，離子最少 → 電導率最低 → 限度最嚴（2.1–2.4 µS/cm）
  • pH > 6.6（偏鹼性）：OH⁻ 離子濃度上升 → 電導率再次升高 → 限度再次放寬

這個「V 形」體現了離子化學的基本原理：純水在 pH 6.0 左右（25°C 下 CO₂ 平衡後）具有最低的本徵電導率。

比喻說明 Analogy

Stage 3 的 pH-電導率關係就像「最佳打擊區」——pH 6.0–6.6 是「甜蜜點」，純水在此範圍離子最少；往酸性或鹼性兩個方向偏移，都意味著有多餘的 H⁺ 或 OH⁻ 離子，電導率自然上升。USP 的 Table 2 就是這種化學關係的精確數學化表達。

重點提示 Key Notes

無菌水的更寬限度：無菌水容許更高的電導率（≤5 µS/cm vs. ~1.3 µS/cm for Bulk Water at 25°C），原因是：

  • 無菌水在容器中已充分吸收 CO₂，電導率本身較高
  • 容器材料（玻璃/塑料）有離子滲出，使電導率增加
  • 小容器（≤10 mL）面積/體積比更大，滲出更多 → 限度更寬（25 µS/cm）

「開瓶前劇烈振搖」的必要性：均勻化容器底部可能沉積的滲出物，確保取到代表性樣品，避免採樣偏差。

無菌水 vs. 散裝水電導率限度對比

散裝水（Bulk Water，25°C, Stage 1）：
  電導率限度：1.3 µS/cm（純理論值）
  適用：WFI 系統、純化水系統線上監測

散裝水（Stage 2，大氣 CO₂ 平衡後）：
  電導率限度：2.1 µS/cm
  適用：離線確認測試

無菌水（成品容器測試，25°C, CO₂ 平衡後）：
  ≤ 10 mL 容器：≤ 25 µS/cm
  > 10 mL 容器：≤ 5 µS/cm

差異倍數：
  5 µS/cm / 1.3 µS/cm ≈ 3.8 倍
  25 µS/cm / 1.3 µS/cm ≈ 19 倍
  （反映容器滲出物的額外貢獻）

實務應用 Practical Application

在 CDMO 的無菌製造驗證中，電導率測試是容器/封口完整性驗證 (container-closure integrity) 的間接指標之一。若同一批次 SWFI（無菌注射用水）的電導率在不同容器間出現顯著差異（如部分容器 3 µS/cm，部分 18 µS/cm），可能提示容器封口問題、批次混淆，或特定批次的玻璃品質異常，需立即啟動偏差調查並隔離受影響樣品。

本章重點回顧 Chapter Summary

  • 電導率原理：離子導電；κ = Θ/R；電池常數 Θ 誤差須 ≤ ±2%
  • 主要干擾離子：Cl⁻（0.47 ppm 模型）、NH₄⁺（0.3 ppm 限度）；CO₂ 溶解為本徵影響
  • 散裝水三階段設計邏輯：Stage 1（快速線上）→ Stage 2（CO₂ 平衡，25°C）→ Stage 3（pH 確認）
  • Stage 1 規定：非溫度補償；查表不得內插；25°C 限度 1.3 µS/cm
  • Stage 2 限度：2.1 µS/cm（所有散裝水在大氣 CO₂ 平衡後）
  • Stage 3 條件：加 KCl 測 pH；pH 須在 5.0–7.0；pH 6.0–6.6 為最低電導率區域
  • 無菌水程序：開瓶前振搖；25°C 平衡至 < 0.1 µS/cm/5 min 變化
  • 無菌水限度：≤10 mL 容器 25 µS/cm；>10 mL 容器 5 µS/cm
  • 溫度準確度：所有測試溫度量測精度須 ±2°C
