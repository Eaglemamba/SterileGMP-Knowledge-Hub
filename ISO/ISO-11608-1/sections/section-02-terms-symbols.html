Section 5.1–5.2: System Designations — NIS 的六大設計分類

原文內容 Original Text

5.1 General Requirements

The NIS shall be designated according to 5.2. These designations form the basis for selecting the appropriate test requirements in Section 9 (Test Matrix).

5.2 System Designations

NISs shall be designated as follows, depending on the container type and dose configuration:

Designation A — Replaceable Container, Multi-Dose, Variable

Needle-based injection device with replaceable container. Each container holds multiple doses, the size of which may be fixed or variable (pre-set by the user).

Example: Insulin pen injector with replaceable cartridge

Designation B1 — Replaceable Container, Single Dose (Full Volume)

Needle-based injection device with replaceable container. Each container holds a single dose, whereby the entire deliverable volume is expelled.

Example: Prefilled syringe-based device, full-expulsion

Designation B2 — Replaceable Container, Single Dose (Partial Volume)

Needle-based injection device with replaceable container. Each container holds a single dose, whereby a portion of the deliverable volume is expelled.

Designation C — Integrated Non-Replaceable Container, Multi-Dose

Needle-based injection device with integrated non-replaceable container. Multiple doses, fixed or variable.

Example: Disposable insulin pen (non-refillable)

Designation D1 — Integrated Container, Single Dose (Full Volume)

Needle-based injection device with integrated non-replaceable container. Single dose, entire deliverable volume expelled.

Example: Most autoinjectors with prefilled PFS

Designation D2 — Integrated Container, Single Dose (Partial Volume)

Needle-based injection device with integrated non-replaceable container. Single dose, portion of volume expelled.

Special requirement: shall be designed so that it is impossible to deliver remaining volume after actuation, and impossible to reactivate the device.

導師解析 Tutorial Commentary

CDMO 最常見：D1 系統

在 CDMO 無菌充填業務中，絕大多數 autoinjector 產品屬於 D1 分類：

  • 整合式容器（PFS 固定在裝置內，不可替換）
  • 單劑量（每次使用一支）
  • 全量釋出（0.5 mL 或 1 mL，完全打出）

D1 系統的劑量準確性評估使用 Dose Delivery Efficiency（剩餘量/充填量的計算），而非固定劑量的上下限判定。

這意味著 CDMO 的充填精度（uniformity of fill）直接影響 D1 系統的 Dose Delivery Efficiency 合規性。

比喻說明：A vs D1 的差異

Designation A（胰島素筆型注射器）就像咖啡膠囊機：機器留在手上，用完換膠囊，每次可選不同份量。

Designation D1（自動注射器）就像一次性打火機：整體不可拆，按一次打完，打完即丟。

兩者的劑量準確性測試邏輯完全不同——前者需要測試「每一個設定劑量的精準度」，後者測試「能否把整瓶藥完全打出」。

練習思考 Practice Questions

  1. 一個 adalimumab biosimilar 的 40 mg/0.8 mL 自動注射器，採用 PFS 固定整合在裝置內，一次性使用。請問其 System Designation 為何？
  2. 同一客戶另有一個可替換 PFS 版本的筆型注射器，可預設 0.4 mL 或 0.8 mL 劑量。System Designation 為何？
  3. D2 系統要求「無法重新啟動」，這個設計要求對 CDMO 充填製程有什麼影響？

Section 4: 統計符號深度解析 — 劑量準確性的數學基礎

原文內容 Original Text

Key Statistical Symbols

Key Equations

Volume from gravimetric measurement:

Vmeas = Gmeas / ρ   (Equation 1)

Specification limits (dual-mode):

  • If Vset ≤ TP: U = Vset + α; L = Vset − α
  • If Vset > TP: U = Vset × (1 + β/100); L = Vset × (1 − β/100)

Acceptance criterion (two-sided):

  • x̄ + k·s ≤ U and x̄ − k·s ≥ L → Pass
  • where k ≥ ktar from Annex B

kact calculation (two-sided):

kact = min[ (U − x̄)/s, (x̄ − L)/s ]

導師解析 Tutorial Commentary

劑量準確性統計邏輯：從樣本到母群體

ISO 11608-1 的劑量準確性驗證使用統計容許區間（Tolerance Interval）方法，而非簡單的「樣本平均值 ± 3σ」。

核心邏輯：

  • 我們測量 n 支 NIS（每支做 R 個 replicate）
  • 計算 x̄（平均值）和 s（標準差）
  • 用 k 值擴大信心區間，確保 至少 95% 的母群體（p=0.95）落在規格內
  • n 越小 → k 越大（不確定性補償），測試越難通過

公式解讀：α 與 β 的切換邏輯

ISO 11608-1 採用「雙段式誤差規格」設計：

小劑量（V ≤ TP）：使用絕對誤差 ±α mL
  例：V_set = 0.1 mL，α = 0.02 mL
  允許範圍：0.08 ~ 0.12 mL（±20%）

大劑量（V > TP）：使用相對誤差 ±β%
  例：V_set = 1.0 mL，β = 5%
  允許範圍：0.95 ~ 1.05 mL（±5%）

轉換點 TP = α / (β/100)

這個設計反映了計量學事實：小劑量的測量精度以絕對值表達更合理；大劑量以百分比更合理。

比喻說明：k 值的直覺理解

想像你要訂購一批杯子，要求容量在 250 ± 10 mL 之間。你只能測量 5 個樣本。

如果 5 個樣本都在 248-252 mL（很均勻），你對整批貨物的信心很高。

但如果 5 個樣本在 240-260 mL（很分散），即使平均值是 250，你不敢保證整批都合格。

k 值就是這個「分散程度」的補償係數——s 大 → 需要更大的 k 才能保證同樣比例的合格率。Annex B 的大表就是預先算好不同 n 和 p 組合下的 k 值。

Gauge R&R 要求：測試設備的精度門檻

Section 6.1 要求：Gauge R&R ≤ 20% of tolerance range（破壞性測試為 30%）。

對 CDMO QC 的含義：

  • 如果容許誤差範圍是 ±0.05 mL（total 0.10 mL），測試系統的 R&R 不得超過 0.02 mL
  • 客戶提交的 DV 報告應包含 Gauge R&R 研究，CDMO 在收到設備時需驗證
  • 重力法（gravimetric）比體積法精度更高，這也是為何 ISO 11608-1 明確要求記錄 Gmeas 並換算為 Vmeas

練習思考 Practice Questions

  1. 某 NIS 的 Vmin = 0.2 mL，Vmax = 1.0 mL，絕對誤差 α = 0.04 mL，相對誤差 β = 4%。計算轉換點 TP = ?（提示：TP = α / (β/100)）
  2. 在 Vset = 0.5 mL（大於 TP）的情況下，上下規格限 U 和 L 各為多少？
  3. 測試 Y=20 支 NIS，每支測 3 個 replicate（N=3）。若 x̄ = 0.495 mL，s = 0.008 mL，kact = (0.524 - 0.495)/0.008 = 3.625。從 Annex B 查 n=60（20×3）、p=0.95 的 ktar ≈ 1.95。此批是否合格？

本節重點回顧 Section Summary

  • 六大系統設計分類：容器可替換性 × 單/多劑量 × 全量/部分釋出 → 決定測試矩陣；CDMO 最常見的 autoinjector 屬 D1（整合式、單劑量、全量）
  • 雙段式誤差規格：小劑量用絕對誤差 ±α，大劑量用相對誤差 ±β%，轉換點 TP = α/(β/100)
  • 統計容許區間方法：x̄ + k·s ≤ U 且 x̄ − k·s ≥ L；k 值從 Annex B 依 n 和 p 查表，補償小樣本的統計不確定性
  • Gauge R&R ≤ 20%：測試設備精度要求，確保測試結果的不確定性不主導合規判定；重力法是 ISO 11608-1 的推薦測量方法
