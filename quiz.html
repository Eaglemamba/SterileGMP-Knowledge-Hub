How to Use the Quiz System

1
Open Claude Code and run /quiz
/quiz PICS-Annex15 /quiz topics/APS-filling-isolator

Claude will read the knowledge MD for that document and generate 10 questions. Questions are asked one at a time — answer each before the next appears.

2
Complete the exam

For each question, Claude provides immediate feedback and a score. After question 10, a summary table and JSON result block are shown.

3
Copy the JSON and record here

Copy the JSON block from Claude's output, click the Record Result tab, paste, and click Save. Your score is saved to browser localStorage — no server required.
